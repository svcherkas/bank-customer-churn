Bank Customer Churn Analysis Dashboard

Этот дашборд, созданный в Power BI, предоставляет комплексный анализ оттока клиентов банка. Он визуализирует ключевые метрики и тенденции, помогающие понять факторы, влияющие на уход клиентов.

Основные характеристики
Общая статистика: Отображает общее количество клиентов, процент ушедших клиентов и процент оставшихся.
Демографический анализ: Показывает распределение клиентов по возрасту, полу и географии.
Кредитный скоринг: Визуализирует распределение кредитных оценок клиентов.
Географический анализ: Представляет распределение клиентов по странам (Франция, Германия, Испания).
Анализ оттока: Сравнивает характеристики ушедших и оставшихся клиентов.

Интерактивные функции
Фильтры: Дашборд включает фильтры по географии, количеству продуктов, полу и статусу активности клиента, позволяющие детально исследовать данные.
Динамические обновления: Все визуализации обновляются в реальном времени при применении фильтров.

Ключевые метрики
Общее количество клиентов: 10,000
Процент ушедших клиентов: 20.4%
Средний возраст клиентов: 38.9 лет
Средний кредитный скор: 650.5
Средний срок обслуживания: 5 лет

Как использовать
Откройте файл дашборда в Power BI Desktop или Power BI Service.
Используйте фильтры в правой части дашборда для сегментации данных.
Наведите курсор на различные элементы для получения дополнительной информации.
Исследуйте взаимосвязи между различными факторами и оттоком клиентов.

Выводы
Этот дашборд позволяет выявить ключевые факторы, влияющие на отток клиентов, такие как географическое положение, возраст, кредитный рейтинг и активность клиента. Он предоставляет ценную информацию для разработки стратегий удержания клиентов и улучшения банковских услуг.

Технические детали
Создано с использованием: Power BI Desktop
Источник данных: CSV файл с информацией о клиентах банка
